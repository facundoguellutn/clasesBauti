# Ejercicios Básicos de Funciones Cuadráticas

Estos ejercicios están diseñados para afianzar los conceptos más fundamentales. Intenta resolverlos y luego comprueba tus respuestas en el archivo de soluciones.

**ID de Ejercicio:** EB01 - EB20

---

### Bloque 1: Identificar Coeficientes y Concavidad

Para cada una de las siguientes funciones, identifica los coeficientes `a`, `b` y `c`, y determina si la parábola abre hacia arriba o hacia abajo.

1.  **EB01:** $f(x) = 2x^2 + 5x - 3$
2.  **EB02:** $f(x) = -x^2 + 4x$
3.  **EB03:** $f(x) = x^2 - 9$
4.  **EB04:** $f(x) = 7 - 3x - x^2$
5.  **EB05:** $f(x) = \frac{1}{2}x^2 + 1$

---

### Bloque 2: Encontrar el Vértice

Calcula las coordenadas del vértice $(h, k)$ para las siguientes funciones.

6.  **EB06:** $f(x) = x^2 - 6x + 8$
7.  **EB07:** $f(x) = -x^2 - 2x + 5$
8.  **EB08:** $f(x) = 2x^2 + 8x$
9.  **EB09:** $f(x) = x^2 - 16$
10. **EB10:** $f(x) = 3(x-2)^2 + 1$ (¡Lee con atención!)

---

### Bloque 3: Encontrar Intersecciones con los Ejes

Para cada función, encuentra el punto de intersección con el eje Y y los puntos de intersección con el eje X (raíces), si existen.

11. **EB11:** $f(x) = x^2 - 4x + 3$
12. **EB12:** $f(x) = x^2 - 25$
13. **EB13:** $f(x) = x^2 + x - 6$
14. **EB14:** $f(x) = -x^2 + 4$
15. **EB15:** $f(x) = x^2 + 4x + 4$

---

### Bloque 4: Convertir entre Formas

16. **EB16:** Convierte $f(x) = x^2 + 8x + 15$ a la forma factorizada.
17. **EB17:** Convierte $f(x) = x^2 - 10x + 20$ a la forma canónica.
18. **EB18:** Convierte $f(x) = 2(x-3)(x+1)$ a la forma general.
19. **EB19:** Convierte $f(x) = 3(x-1)^2 + 5$ a la forma general.
20. **EB20:** Dada la función con raíces en $x=5$ y $x=-2$, y que pasa por el punto (1, -24), encuentra su forma general. (Pista: empieza con la forma factorizada $f(x) = a(x-r_1)(x-r_2)$ y usa el punto para encontrar `a`).
