## Soluciones de Ejercicios Básicos (EB01 - EB20)

**EB01:** $f(x) = 2x^2 + 5x - 3$
*   $a=2, b=5, c=-3$.
*   Concavidad: Hacia arriba ($a>0$).

**EB02:** $f(x) = -x^2 + 4x$
*   $a=-1, b=4, c=0$.
*   Concavidad: Hacia abajo ($a<0$).

**EB03:** $f(x) = x^2 - 9$
*   $a=1, b=0, c=-9$.
*   Concavidad: Hacia arriba ($a>0$).

**EB04:** $f(x) = 7 - 3x - x^2$ (reordenando: $-x^2 - 3x + 7$)
*   $a=-1, b=-3, c=7$.
*   Concavidad: Hacia abajo ($a<0$).

**EB05:** $f(x) = \frac{1}{2}x^2 + 1$
*   $a=1/2, b=0, c=1$.
*   Concavidad: Hacia arriba ($a>0$).

**EB06:** $f(x) = x^2 - 6x + 8$
*   $h = -(-6)/(2 \cdot 1) = 3$.
*   $k = f(3) = 3^2 - 6(3) + 8 = 9 - 18 + 8 = -1$.
*   Vértice: **(3, -1)**.

**EB07:** $f(x) = -x^2 - 2x + 5$
*   $h = -(-2)/(2 \cdot -1) = 2/(-2) = -1$.
*   $k = f(-1) = -(-1)^2 - 2(-1) + 5 = -1 + 2 + 5 = 6$.
*   Vértice: **(-1, 6)**.

**EB08:** $f(x) = 2x^2 + 8x$
*   $h = -8/(2 \cdot 2) = -2$.
*   $k = f(-2) = 2(-2)^2 + 8(-2) = 8 - 16 = -8$.
*   Vértice: **(-2, -8)**.

**EB09:** $f(x) = x^2 - 16$
*   $h = -0/(2 \cdot 1) = 0$.
*   $k = f(0) = -16$.
*   Vértice: **(0, -16)**.

**EB10:** $f(x) = 3(x-2)^2 + 1$
*   Forma canónica. Se lee directamente.
*   Vértice: **(2, 1)**.

**EB11:** $f(x) = x^2 - 4x + 3$
*   Corte Y: $(0, 3)$.
*   Raíces: $(x-1)(x-3)=0 \implies x=1, x=3$. Puntos: **(1, 0) y (3, 0)**.

**EB12:** $f(x) = x^2 - 25$
*   Corte Y: $(0, -25)$.
*   Raíces: $x^2=25 \implies x=\pm 5$. Puntos: **(5, 0) y (-5, 0)**.

**EB13:** $f(x) = x^2 + x - 6$
*   Corte Y: $(0, -6)$.
*   Raíces: $(x+3)(x-2)=0 \implies x=-3, x=2$. Puntos: **(-3, 0) y (2, 0)**.

**EB14:** $f(x) = -x^2 + 4$
*   Corte Y: $(0, 4)$.
*   Raíces: $-x^2+4=0 \implies x^2=4 \implies x=\pm 2$. Puntos: **(2, 0) y (-2, 0)**.

**EB15:** $f(x) = x^2 + 4x + 4$
*   Corte Y: $(0, 4)$.
*   Raíces: $(x+2)^2=0 \implies x=-2$ (raíz doble). Punto: **(-2, 0)**.

**EB16:** $f(x) = x^2 + 8x + 15$
*   Dos números que multiplican 15 y suman 8 son 3 y 5.
*   Forma factorizada: **$f(x) = (x+3)(x+5)$**.

**EB17:** $f(x) = x^2 - 10x + 20$
*   $h = -(-10)/2 = 5$.
*   $k = f(5) = 5^2 - 10(5) + 20 = 25 - 50 + 20 = -5$.
*   Forma canónica: **$f(x) = (x-5)^2 - 5$**.

**EB18:** $f(x) = 2(x-3)(x+1)$
*   $f(x) = 2(x^2 + x - 3x - 3) = 2(x^2 - 2x - 3)$.
*   Forma general: **$f(x) = 2x^2 - 4x - 6$**.

**EB19:** $f(x) = 3(x-1)^2 + 5$
*   $f(x) = 3(x^2 - 2x + 1) + 5 = 3x^2 - 6x + 3 + 5$.
*   Forma general: **$f(x) = 3x^2 - 6x + 8$**.

**EB20:** Raíces 5 y -2, pasa por (1, -24).
*   Forma factorizada: $f(x) = a(x-5)(x+2)$.
*   Sustituir (1, -24): $-24 = a(1-5)(1+2) \implies -24 = a(-4)(3) \implies -24 = -12a \implies a=2$.
*   $f(x) = 2(x-5)(x+2) = 2(x^2 - 3x - 10)$.
*   Forma general: **$f(x) = 2x^2 - 6x - 20$**.

## Soluciones de Ejercicios Intermedios (EI01 - EI20)

**EI01:** $f(x) = 2x^2 - 5x - 3$
*   $\Delta = (-5)^2 - 4(2)(-3) = 25 + 24 = 49 > 0$. **Dos raíces reales distintas**.
*   $x = (5 \pm \sqrt{49})/4 = (5 \pm 7)/4$. Raíces: **$x_1 = 3, x_2 = -1/2$**.

**EI02:** $f(x) = 9x^2 - 6x + 1$
*   $\Delta = (-6)^2 - 4(9)(1) = 36 - 36 = 0$. **Una raíz real doble**.
*   $x = 6/18 = 1/3$. Raíz: **$x = 1/3$**.

**EI03:** $f(x) = x^2 + 3x + 5$
*   $\Delta = 3^2 - 4(1)(5) = 9 - 20 = -11 < 0$. **Sin raíces reales** (dos complejas).

**EI04:** $f(x) = -3x^2 + x + 1$
*   $\Delta = 1^2 - 4(-3)(1) = 1 + 12 = 13 > 0$. **Dos raíces reales distintas**.
*   $x = (-1 \pm \sqrt{13})/-6$. Raíces: **$x_1 = (1 - \sqrt{13})/6, x_2 = (1 + \sqrt{13})/6$**.

**EI05:** $f(x) = 2x^2 - 12x + 10$
*   $f(x) = 2(x^2 - 6x) + 10 = 2(x^2 - 6x + 9 - 9) + 10 = 2((x-3)^2 - 9) + 10$.
*   $f(x) = 2(x-3)^2 - 18 + 10$. Forma canónica: **$f(x) = 2(x-3)^2 - 8$**.

**EI06:** $f(x) = -x^2 + 5x - 6$
*   $f(x) = -(x^2 - 5x) - 6 = -(x^2 - 5x + (5/2)^2 - (5/2)^2) - 6$.
*   $f(x) = -((x - 5/2)^2 - 25/4) - 6 = -(x-5/2)^2 + 25/4 - 24/4$.
*   Forma canónica: **$f(x) = -(x-5/2)^2 + 1/4$**.

**EI07:** $3x^2 + 10x - 8$
*   $ac = -24$. Dos números que multiplican -24 y suman 10 son 12 y -2.
*   $3x^2 + 12x - 2x - 8 = 3x(x+4) - 2(x+4)$.
*   Factorización: **$(3x-2)(x+4)$**.

**EI08:** $4x^2 - 20x + 25$
*   $\Delta = (-20)^2 - 4(4)(25) = 400 - 400 = 0$. Es un **Trinomio Cuadrado Perfecto**.
*   Factorización: **$(2x-5)^2$**.

**EI09:** $x^2 - 2x + 1 = x + 1 \implies x^2 - 3x = 0 \implies x(x-3)=0$.
*   $x=0 \implies y=1$. Punto: **(0, 1)**.
*   $x=3 \implies y=4$. Punto: **(3, 4)**.

**EI10:** $-x^2 + 5 = -x + 3 \implies x^2 - x - 2 = 0 \implies (x-2)(x+1)=0$.
*   $x=2 \implies y=1$. Punto: **(2, 1)**.
*   $x=-1 \implies y=4$. Punto: **(-1, 4)**.

**EI11:** $x^2 = -x^2 + 4x \implies 2x^2 - 4x = 0 \implies 2x(x-2)=0$.
*   $x=0 \implies y=0$. Punto: **(0, 0)**.
*   $x=2 \implies y=4$. Punto: **(2, 4)**.

**EI12:** $x^2 + 1 = x \implies x^2 - x + 1 = 0$.
*   $\Delta = (-1)^2 - 4(1)(1) = -3 < 0$.
*   **No hay puntos de intersección**.

**EI13:** $f(x) = x^2 - 8x + 12$
*   Arriba. Vértice: $(4, -4)$. Corte Y: $(0, 12)$. Raíces: $x=2, x=6$.

**EI14:** $f(x) = -2x^2 - 4x + 6$
*   Abajo. Vértice: $(-1, 8)$. Corte Y: $(0, 6)$. Raíces: $x=1, x=-3$.

**EI15:** $f(x) = (x-3)^2$
*   Arriba. Vértice: $(3, 0)$. Corte Y: $(0, 9)$. Raíz: $x=3$ (doble).

**EI16:** Vértice (2, -9), pasa por (5, 0).
*   $f(x) = a(x-2)^2 - 9$.
*   $0 = a(5-2)^2 - 9 \implies 9 = 9a \implies a=1$.
*   Ecuación: **$f(x) = (x-2)^2 - 9$** o **$f(x) = x^2 - 4x - 5$**.

**EI17:** Raíces -1 y 3, corte Y (0, -6).
*   $f(x) = a(x+1)(x-3)$.
*   $-6 = a(0+1)(0-3) \implies -6 = -3a \implies a=2$.
*   $f(x) = 2(x+1)(x-3) = 2(x^2-2x-3)$.
*   Ecuación: **$f(x) = 2x^2 - 4x - 6$**.

**EI18:** Una raíz doble $\implies \Delta=0$.
*   $k^2 - 4(1)(9) = 0 \implies k^2 = 36 \implies k = \pm 6$.
*   Valores de k: **6 y -6**.

**EI19:** $a=-2 < 0$, la parábola abre hacia abajo. Por lo tanto, el vértice es un **máximo**.

**EI20:** $f(x) = 2(x-1)(x+3)$
*   Raíces: 1 y -3. Vértice $h = (1-3)/2 = -1$.
*   $k = f(-1) = 2(-1-1)(-1+3) = 2(-2)(2) = -8$.
*   Forma canónica: **$f(x) = 2(x+1)^2 - 8$**.
