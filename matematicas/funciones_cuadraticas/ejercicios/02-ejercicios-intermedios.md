# Ejercicios Intermedios de Funciones Cuadráticas

Estos ejercicios requieren combinar varios conceptos. Deberás elegir el método más adecuado para resolver cada problema.

**ID de Ejercicio:** EI01 - EI26

---

### Bloque 1: Clasificación por Discriminante y Resolución

Para cada función, calcula el discriminante, determina el número y tipo de raíces, y luego encuéntralas (si son reales).

1.  **EI01:** $f(x) = 2x^2 - 5x - 3$
2.  **EI02:** $f(x) = 9x^2 - 6x + 1$
3.  **EI03:** $f(x) = x^2 + 3x + 5$
4.  **EI04:** $f(x) = -3x^2 + x + 1$

---

### Bloque 2: Completar el Cuadrado y Factorización con $a \neq 1$

5.  **EI05:** Convierte $f(x) = 2x^2 - 12x + 10$ a la forma canónica completando el cuadrado.
6.  **EI06:** Convierte $f(x) = -x^2 + 5x - 6$ a la forma canónica.
7.  **EI07:** Factoriza la expresión $3x^2 + 10x - 8$ usando el método de agrupación (AC).
8.  **EI08:** Factoriza la expresión $4x^2 - 20x + 25$. ¿Qué caso especial es?

---

### Bloque 3: Intersección entre Funciones

Encuentra los puntos de intersección $(x, y)$ para los siguientes pares de funciones.

9.  **EI09:** La parábola $f(x) = x^2 - 2x + 1$ y la recta $g(x) = x + 1$.
10. **EI10:** La parábola $f(x) = -x^2 + 5$ y la recta $g(x) = -x + 3$.
11. **EI11:** Las parábolas $f(x) = x^2$ y $g(x) = -x^2 + 4x$.
12. **EI12:** La parábola $f(x) = x^2 + 1$ y la recta $g(x) = x$. ¿Qué ocurre aquí?

---

### Bloque 4: Bosquejo de Gráficas

Para cada función, encuentra la concavidad, el vértice, los cortes con los ejes y bosqueja la gráfica a mano.

13. **EI13:** $f(x) = x^2 - 8x + 12$
14. **EI14:** $f(x) = -2x^2 - 4x + 6$
15. **EI15:** $f(x) = (x-3)^2$

---

### Bloque 5: Problemas Mixtos

16. **EI16:** Encuentra la ecuación de una parábola que tiene su vértice en $(2, -9)$ y pasa por el punto $(5, 0)$. (Pista: usa la forma canónica).
17. **EI17:** Una parábola tiene raíces en $x=-1$ y $x=3$. Su intersección con el eje Y es en $(0, -6)$. Encuentra su forma general.
18. **EI18:** Para la función $f(x) = x^2 + kx + 9$, ¿qué valor de `k` hará que la función tenga exactamente una raíz real doble? (Pista: ¿qué debe valer el discriminante?).
19. **EI19:** El eje de simetría de la parábola $f(x) = -2x^2 + 12x - 5$ es $x=3$. Sin calcular el vértice completo, ¿la parábola tiene un máximo o un mínimo y por qué?
20. **EI20:** Reescribe la función $f(x) = 2(x-1)(x+3)$ en forma canónica. (Pista: puedes pasar primero por la forma general, o encontrar el vértice a partir de las raíces).

---

### Bloque 6: Representación Gráfica y Análisis Completo

Para cada una de las siguientes funciones, realiza lo siguiente:
- Representa gráficamente la función
- Expresa de qué modo se presenta el gráfico (forma canónica, factorizada o polinómica)
- Completa la tabla con: $x_1$, $x_2$ (raíces), intervalo de crecimiento, intervalo de decrecimiento, máximo, mínimo, intervalo de positividad e intervalo de negatividad

21. **EI21:** $f(x) = 2(x + 1)(x + \frac{7}{2})$

22. **EI22:** $g(x) = -\frac{1}{4}x^2 - x + 3$

23. **EI23:** $h(x) = \frac{1}{2}(x - 3)^2 - 2$

24. **EI24:** $f(x) = -2(x + 1)(x + 3)$

25. **EI25:** $g(x) = -x^2 - 2x + 3$

26. **EI26:** $h(x) = -\frac{1}{2}(x + 3)^2 + 2$
