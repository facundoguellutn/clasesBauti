# 09 - Atajos Básicos de Teclado en Visual Studio Code

## ¿Qué son los atajos de teclado?

Los **atajos de teclado** (también llamados "shortcuts" o "hotkeys") son combinaciones de teclas que realizan acciones rápidamente sin necesidad de usar el mouse.

## ¿Por qué usar atajos?

- **Más rápido**: No necesitas mover el mouse
- **Más eficiente**: Ahorras tiempo
- **Más profesional**: Los programadores usan atajos constantemente
- **Menos interrupciones**: Mantienes las manos en el teclado

## Notación de atajos

En este documento usaremos esta notación:
- `Ctrl` = Tecla Control (Windows/Linux)
- `Cmd` = Tecla Command (Mac)
- `Shift` = Tecla Mayús
- `Alt` = Tecla Alt (Windows/Linux)
- `Option` = Tecla Option (Mac)
- `+` = Presiona ambas teclas al mismo tiempo

**Ejemplo**: `Ctrl + S` significa "presiona Control y S al mismo tiempo"

## Atajos esenciales (los más importantes)

### Guardar archivo
- **Windows/Linux**: `Ctrl + S`
- **Mac**: `Cmd + S`
- **¿Para qué?**: Guarda el archivo actual. ¡Úsalo frecuentemente!

### Abrir archivo
- **Windows/Linux**: `Ctrl + O`
- **Mac**: `Cmd + O`
- **¿Para qué?**: Abre un archivo existente

### Nuevo archivo
- **Windows/Linux**: `Ctrl + N`
- **Mac**: `Cmd + N`
- **¿Para qué?**: Crea un archivo nuevo

### Cerrar archivo
- **Windows/Linux**: `Ctrl + W`
- **Mac**: `Cmd + W`
- **¿Para qué?**: Cierra el archivo actual

### Cerrar VS Code
- **Windows/Linux**: `Ctrl + Q` o `Alt + F4`
- **Mac**: `Cmd + Q`
- **¿Para qué?**: Cierra VS Code completamente

## Atajos de edición

### Deshacer
- **Windows/Linux**: `Ctrl + Z`
- **Mac**: `Cmd + Z`
- **¿Para qué?**: Deshace la última acción

### Rehacer
- **Windows/Linux**: `Ctrl + Y` o `Ctrl + Shift + Z`
- **Mac**: `Cmd + Shift + Z`
- **¿Para qué?**: Rehace lo que deshiciste

### Copiar
- **Windows/Linux**: `Ctrl + C`
- **Mac**: `Cmd + C`
- **¿Para qué?**: Copia el texto seleccionado

### Cortar
- **Windows/Linux**: `Ctrl + X`
- **Mac**: `Cmd + X`
- **¿Para qué?**: Corta el texto seleccionado

### Pegar
- **Windows/Linux**: `Ctrl + V`
- **Mac**: `Cmd + V`
- **¿Para qué?**: Pega el texto copiado

### Seleccionar todo
- **Windows/Linux**: `Ctrl + A`
- **Mac**: `Cmd + A`
- **¿Para qué?**: Selecciona todo el contenido del archivo

### Buscar en el archivo
- **Windows/Linux**: `Ctrl + F`
- **Mac**: `Cmd + F`
- **¿Para qué?**: Busca texto en el archivo actual

### Reemplazar
- **Windows/Linux**: `Ctrl + H`
- **Mac**: `Cmd + Option + F` o `Cmd + H`
- **¿Para qué?**: Busca y reemplaza texto

### Buscar en todos los archivos
- **Windows/Linux**: `Ctrl + Shift + F`
- **Mac**: `Cmd + Shift + F`
- **¿Para qué?**: Busca texto en todos los archivos del proyecto

## Atajos de navegación

### Ir a línea
- **Windows/Linux**: `Ctrl + G`
- **Mac**: `Cmd + G`
- **¿Para qué?**: Salta a una línea específica (escribe el número)

### Ir a archivo
- **Windows/Linux**: `Ctrl + P`
- **Mac**: `Cmd + P`
- **¿Para qué?**: Abre rápidamente un archivo escribiendo su nombre

### Ir a símbolo
- **Windows/Linux**: `Ctrl + Shift + O`
- **Mac**: `Cmd + Shift + O`
- **¿Para qué?**: Salta a una función o variable en el archivo actual

### Ir a definición
- **Windows/Linux**: `F12`
- **Mac**: `F12`
- **¿Para qué?**: Ve a donde está definida una función o variable

### Volver atrás
- **Windows/Linux**: `Alt + ←` (flecha izquierda)
- **Mac**: `Ctrl + -`
- **¿Para qué?**: Vuelve a donde estabas antes

### Ir adelante
- **Windows/Linux**: `Alt + →` (flecha derecha)
- **Mac**: `Ctrl + Shift + -`
- **¿Para qué?**: Avanza después de volver atrás

## Atajos de interfaz

### Mostrar/ocultar barra lateral
- **Windows/Linux**: `Ctrl + B`
- **Mac**: `Cmd + B`
- **¿Para qué?**: Oculta o muestra la barra lateral izquierda

### Mostrar/ocultar terminal
- **Windows/Linux**: `Ctrl + Ñ` o `` Ctrl + ` ``
- **Mac**: `Ctrl + Ñ` o `` Cmd + ` ``
- **¿Para qué?**: Abre o cierra el terminal integrado

### Mostrar/ocultar panel inferior
- **Windows/Linux**: `Ctrl + J`
- **Mac**: `Cmd + J`
- **¿Para qué?**: Muestra u oculta el panel inferior (terminal, problemas, etc.)

### Vista de solo editor (Zen Mode)
- **Windows/Linux**: `Ctrl + K` luego `Z`
- **Mac**: `Cmd + K` luego `Z`
- **¿Para qué?**: Oculta todo excepto el editor (modo zen)

### Salir de Zen Mode
- **Windows/Linux/Mac**: `Esc` dos veces

## Atajos de código

### Comentar/descomentar línea
- **Windows/Linux**: `Ctrl + /`
- **Mac**: `Cmd + /`
- **¿Para qué?**: Comenta o descomenta la línea actual

### Formatear documento
- **Windows/Linux**: `Shift + Alt + F`
- **Mac**: `Shift + Option + F`
- **¿Para qué?**: Formatea (organiza) todo el código del archivo

### Duplicar línea
- **Windows/Linux**: `Shift + Alt + ↓` o `Shift + Alt + ↑`
- **Mac**: `Shift + Option + ↓` o `Shift + Option + ↑`
- **¿Para qué?**: Duplica la línea actual arriba o abajo

### Mover línea
- **Windows/Linux**: `Alt + ↓` o `Alt + ↑`
- **Mac**: `Option + ↓` o `Option + ↑`
- **¿Para qué?**: Mueve la línea actual arriba o abajo

### Eliminar línea
- **Windows/Linux**: `Ctrl + Shift + K`
- **Mac**: `Cmd + Shift + K`
- **¿Para qué?**: Elimina la línea completa donde está el cursor

### Selección múltiple (siguiente ocurrencia)
- **Windows/Linux**: `Ctrl + D`
- **Mac**: `Cmd + D`
- **¿Para qué?**: Selecciona la siguiente ocurrencia de la palabra seleccionada

### Cursor múltiple
- **Windows/Linux**: `Alt + Click`
- **Mac**: `Option + Click`
- **¿Para qué?**: Agrega un cursor donde hagas clic

## Atajos de extensiones y herramientas

### Abrir extensiones
- **Windows/Linux**: `Ctrl + Shift + X`
- **Mac**: `Cmd + Shift + X`
- **¿Para qué?**: Abre el panel de extensiones

### Abrir configuración
- **Windows/Linux**: `Ctrl + ,`
- **Mac**: `Cmd + ,`
- **¿Para qué?**: Abre la configuración de VS Code

### Abrir paleta de comandos
- **Windows/Linux**: `Ctrl + Shift + P`
- **Mac**: `Cmd + Shift + P`
- **¿Para qué?**: Abre un menú donde puedes buscar cualquier comando

## Cómo aprender los atajos

### Método 1: Usa los atajos gradualmente
1. Empieza con los más básicos: `Ctrl + S`, `Ctrl + C`, `Ctrl + V`
2. Agrega uno o dos nuevos cada día
3. Practica hasta que se vuelvan automáticos

### Método 2: Imprime una hoja de referencia
1. Ve a **Help** → **Keyboard Shortcuts Reference**
2. O busca "VS Code keyboard shortcuts cheat sheet" en internet
3. Tenla a mano mientras trabajas

### Método 3: Usa la paleta de comandos
- Presiona `Ctrl + Shift + P` (o `Cmd + Shift + P` en Mac)
- Escribe el nombre de la acción que quieres hacer
- VS Code te mostrará el atajo si existe

## Personalizar atajos

Puedes cambiar los atajos si no te gustan:

1. Ve a **File** → **Preferences** → **Keyboard Shortcuts** (`Ctrl + K` luego `Ctrl + S`)
2. Busca el comando que quieres cambiar
3. Haz doble clic en el atajo actual
4. Presiona la nueva combinación de teclas
5. Presiona Enter para confirmar

## Los 10 atajos más importantes para empezar

1. `Ctrl + S` - Guardar (¡úsalo constantemente!)
2. `Ctrl + F` - Buscar en archivo
3. `Ctrl + P` - Ir a archivo
4. `Ctrl + /` - Comentar línea
5. `Ctrl + B` - Mostrar/ocultar barra lateral
6. `Ctrl + Ñ` - Mostrar/ocultar terminal
7. `Ctrl + Z` - Deshacer
8. `Ctrl + Shift + P` - Paleta de comandos
9. `F12` - Ir a definición
10. `Ctrl + ,` - Configuración

## Tips útiles

✅ **Empieza con pocos**: No intentes aprender todos a la vez
✅ **Practica diariamente**: La práctica hace al maestro
✅ **No te preocupes**: Siempre puedes usar el mouse si olvidas un atajo
✅ **Usa la paleta de comandos**: Es una forma fácil de encontrar comandos
✅ **Personaliza**: Cambia los atajos que no te gusten
