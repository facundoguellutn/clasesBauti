# 03 - Primeros Pasos en Visual Studio Code

## Abrir Visual Studio Code

### Método 1: Desde el menú de inicio/aplicaciones

1. **Windows**: Busca "Visual Studio Code" en el menú de inicio y haz clic
2. **Mac**: Abre "Finder", ve a "Applications" y haz doble clic en "Visual Studio Code"
3. **Linux**: Busca "Code" en el menú de aplicaciones

### Método 2: Desde la terminal (avanzado)

1. Abre la terminal o símbolo del sistema
2. Escribe: `code`
3. Presiona Enter

### Método 3: Abrir una carpeta directamente

1. Navega a una carpeta en tu computadora
2. Haz clic derecho en la carpeta
3. Selecciona "Open with Code" o "Abrir con Code" (si está disponible)

## La pantalla de inicio

Cuando abres VS Code por primera vez, verás:

- **Barra lateral izquierda**: Iconos para diferentes funciones
- **Área central**: Donde escribirás tu código
- **Barra superior**: Menús (Archivo, Editar, Ver, etc.)
- **Barra inferior**: Información y estado

## Crear tu primer archivo

### Paso 1: Crear un archivo nuevo

1. Haz clic en el menú **"File"** (Archivo) en la parte superior
2. Selecciona **"New File"** (Nuevo archivo)
   - **Atajo rápido**: Presiona `Ctrl + N` (Windows/Linux) o `Cmd + N` (Mac)

### Paso 2: Escribir algo

1. En el área de edición (parte central), escribe algo simple:
   ```
   Hola, este es mi primer archivo en VS Code
   ```

### Paso 3: Guardar el archivo

1. Haz clic en **"File"** → **"Save"** (Guardar)
   - **Atajo rápido**: `Ctrl + S` (Windows/Linux) o `Cmd + S` (Mac)
2. Se abrirá un cuadro de diálogo para elegir dónde guardar
3. Elige una ubicación (por ejemplo, tu escritorio o una carpeta de documentos)
4. Escribe un nombre para el archivo, por ejemplo: `mi-primer-archivo.txt`
5. Haz clic en **"Save"** (Guardar)

¡Felicidades! Acabas de crear y guardar tu primer archivo en VS Code.

## Abrir un archivo existente

### Método 1: Desde el menú

1. Haz clic en **"File"** → **"Open File"** (Abrir archivo)
   - **Atajo rápido**: `Ctrl + O` (Windows/Linux) o `Cmd + O` (Mac)
2. Navega hasta el archivo que quieres abrir
3. Selecciona el archivo y haz clic en **"Open"** (Abrir)

### Método 2: Arrastrar y soltar

1. Abre la carpeta donde está tu archivo
2. Arrastra el archivo y suéltalo en la ventana de VS Code

## Trabajar con pestañas

VS Code muestra cada archivo abierto en una pestaña:

- **Abrir múltiples archivos**: Cada archivo aparece como una pestaña en la parte superior
- **Cambiar entre archivos**: Haz clic en la pestaña del archivo que quieres ver
- **Cerrar un archivo**: Haz clic en la "X" de la pestaña o presiona `Ctrl + W` (Windows/Linux) o `Cmd + W` (Mac)

## Ejemplo práctico: Crear un archivo HTML

Vamos a crear un archivo HTML simple:

1. Crea un nuevo archivo (`Ctrl + N` o `Cmd + N`)
2. Escribe esto:
   ```html
   <!DOCTYPE html>
   <html>
   <head>
       <title>Mi primera página</title>
   </head>
   <body>
       <h1>¡Hola desde VS Code!</h1>
       <p>Este es mi primer archivo HTML.</p>
   </body>
   </html>
   ```
3. Guarda el archivo como `mi-pagina.html`
4. Observa cómo VS Code colorea el código automáticamente (esto se llama "resaltado de sintaxis")

## Conceptos importantes

### Archivos sin guardar

- Si un archivo tiene cambios sin guardar, verás un punto blanco en la pestaña
- Siempre guarda tus cambios frecuentemente (`Ctrl + S` o `Cmd + S`)

### Extensiones de archivo

- `.txt` = archivo de texto
- `.html` = página web
- `.js` = JavaScript
- `.py` = Python
- `.css` = estilos
- VS Code reconoce el tipo de archivo por su extensión
