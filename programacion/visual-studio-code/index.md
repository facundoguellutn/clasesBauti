# Bienvenido al Curso de Visual Studio Code

Este conjunto de archivos Markdown forma un curso completo y autocontenido para aprender a usar Visual Studio Code desde cero. El objetivo es proporcionar un recurso claro, estructurado y fácil de seguir para que cualquier persona pueda entender qué es VS Code y empezar a programar.

## Mapa de Lectura Recomendado

Para un aprendizaje progresivo, te recomendamos seguir el orden de los archivos, ya que están numerados para guiarte desde los conceptos más básicos hasta poder empezar a programar.

1. **Conceptos Fundamentales:**
   * `01-introduccion.md`: ¿Qué es Visual Studio Code y para qué sirve?
   * `02-instalacion.md`: Cómo instalar VS Code en tu computadora
   * `03-primeros-pasos.md`: Abrir VS Code y crear tu primer archivo

2. **Interfaz y Navegación:**
   * `04-interfaz-basica.md`: Conoce las partes principales de VS Code
   * `05-escribir-codigo.md`: Cómo escribir código en VS Code
   * `06-ejecutar-codigo.md`: Ejecutar tu código desde VS Code

3. **Funcionalidades Esenciales:**
   * `07-extensiones-basicas.md`: Instalar extensiones útiles
   * `08-terminal-basico.md`: Usar el terminal integrado
   * `09-atajos-basicos.md`: Atajos de teclado para ser más rápido
   * `10-trabajar-con-carpetas.md`: Organizar tus proyectos

4. **Herramientas Útiles:**
   * `11-buscar-y-reemplazar.md`: Buscar texto en tus archivos
   * `12-configuracion-basica.md`: Personalizar VS Code a tu gusto

5. **Recursos Adicionales:**
   * `13-resumen.md`: Resumen de todo lo aprendido
   * `14-faq.md`: Preguntas frecuentes

6. **Ejercicios:**
   * `ejercicios/` - Ejercicios prácticos para practicar

## ¿Cómo Usar Estos Archivos?

* **Lee en orden:** Sigue la numeración para una mejor comprensión.
* **Practica activamente:** Intenta hacer cada paso por tu cuenta.
* **No tengas miedo de experimentar:** VS Code es muy seguro, puedes explorar sin problemas.
* **Haz los ejercicios:** La práctica es fundamental para aprender.

## Requisitos Previos

Para aprovechar al máximo este curso, NO necesitas conocimientos previos de:
* Programación (aprenderás mientras usas VS Code)
* Editores de código
* Herramientas de desarrollo

Solo necesitas:
* Una computadora (Windows, Mac o Linux)
* Conexión a internet (para descargar VS Code y extensiones)
* Ganas de aprender

¡Espero que disfrutes y aprendas mucho!
